STAR WARS: Cantina Games

Description

Made using Unity 2020.1.15f1

Using methods I learned in Data Structures to create a working deck that can be shuffled and dealt, I created this game project originally in Java and ended up porting to Unity as it was easier to create a GUI. I worked on this project on and off for a year until I finally decided I needed to plow through and create a working product.
