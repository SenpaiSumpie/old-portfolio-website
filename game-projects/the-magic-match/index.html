The Magic Match

Description

Made using Game Maker Studio 2 v2.3.4.577

For my CS 413 - Virtual Worlds class we are required to make 4 projects with a team. Our first project's theme was 'Puzzles'. Not wanting to do anything too grand for our first project we decided on doing a simple memory matching game where you spend a turn flipping cards until you make a match. To add our own spin on the game we have it so that the player has a limited amount of moves before the cards get reset. The theme we decided on was using elemental runes and when the player makes a match it ignites an element at the top of the scene view. We had two weeks to complete this project.

I ended up doing all of the artwork and animations, and my main contribution to the programming was the Graphics Programming.
