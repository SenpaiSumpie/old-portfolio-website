Unreal Engine Character UI

Description

Made using Unreal Engine 4 v4.27.0

In a scheme to fix up all of my started game projects, this is a UI demonstration for a game where you have specific characters that do different things to 'win' the game. Made using Unreal Engines Unreal Motion Graphics (UMG). This could be an example for games such as League of Legends, Naraka Bladepoint, and many others. I plan to eventually extend this to making a demo 'in-game UI'.
