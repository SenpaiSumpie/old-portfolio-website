Project Equinox

Description

Made using Unity 2020.3.18f1

I've been planning this for the past couple of years. I've been debating on what types of frameworks I would use in this projects creation. I came up with the idea of a story and have been fine tuning it and adding to it all this time. Currently I am planning on using Unity instead of any other engine frameworks as I own a lot of art assets from the platform.
